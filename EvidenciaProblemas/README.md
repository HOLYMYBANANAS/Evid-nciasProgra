# Evidencias de Problemas LeetCode
Este repositorio contiene mis soluciones a tres ejercicios seleccionados de LeetCode como evidencia de práctica individual de programación, análisis algorítmico y manejo de estructuras de datos. Cada problema incluye su respectivo código, explicación y video demostrativo de la solución.

## Descripción del trabajo realizado
Durante esta actividad resolví tres problemas de dificultad básica e intermedia, aplicando conceptos como manipulación de arreglos, pilas (stack) y recorridos en árboles N-arios. Para cada ejercicio:

- Analicé el planteamiento y los casos borde.
- Implementé una solución correcta y eficiente.
- Probé el código en la plataforma de LeetCode hasta obtener el 100% de aceptación.
- Grabé un video explicando el razonamiento, la lógica aplicada y la complejidad temporal/espacial.

A continuación se listan los problemas trabajados.

## Problemas Resueltos

### 1. Sort Array by Parity: https://leetcode.com/problems/sort-array-by-parity/
Ordena un arreglo moviendo todos los números pares al inicio y los impares al final. Implementé una solución lineal utilizando dos punteros para optimizar el procesamiento.

### 2. Make the String Great: https://leetcode.com/problems/make-the-string-great/
Eliminación de pares de caracteres que difieren solo en mayúsculas/minúsculas. Para resolverlo utilicé una estructura tipo **stack** que permite revisar y limpiar la cadena eficientemente.

### 3. N-ary Tree Postorder Traversal: https://leetcode.com/problems/n-ary-tree-postorder-traversal/
Recorrido *postorder* en un árbol N-ario. Implementé tanto una versión recursiva como una iterativa que utiliza una pila para simular la lógica del recorrido.
